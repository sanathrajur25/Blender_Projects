Steps to Create a Gear
--------------------------------------
Step 1: Place a cylinder (solid - mesh ) by using hot key Shift+A
Step 2 : scale the cylinder to 0.2 
step 3 : jump into edit mode by clicking onto tab button 
Step 4: Select only the top 2 surfaces usinng face select mode
step 5 click on tab again and press ctrl+A select Scale option
Step 6: switch back to edit mode by clicking onto tab 
Step 7 : click i and make smaller radius , repeat until ur satisfied
Step 8 : select 2 rings and add depth og -0.1m
step 9 : select the outside circle and go to select and choose checker deselect option 
Step 10 : add thickness of 0.05 and depth of 0.2 
 Your gear is ready !

------------------------------------------------------------------
To make it rotate 
------------------------------------------------------------------
select the object created and  press N in rotate option go to z axis and input #frame/300
for oposite direction #frame /-300
